# Outstagram
인스타그램 따라하기
## 프로젝트 소개
- 회원가입 및 로그인
- token을 가진 회원만 글 확인 가능
- 전체 포스팅, 나의 포스팅 확인
- 사진과 내용 업로드

## 사용한 기능
- Retrofit
- ViewModel
- Stetho
